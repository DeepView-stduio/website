一、人脸检测接口

类别 数据 类型 说明
接口url http://api.qiansou.cn/api/facedetectv4?appid=sdfsondfadfasdf1 appid注册后免费申请获得
HTTP请求方式 Post
请求参数名 faceimage string 图片base64编码字符串
请求JSON格式
返回内容 facemodels Json数组 人脸模型集合，参考返回json信息
result bool 检测是否成功
errorinfo string 错误提示或提示信息
返回JSON信息
备注 返回的lefteye、righteye、mouth为预留字段，使用的时候请忽略
